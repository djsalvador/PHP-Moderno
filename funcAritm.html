FUNÇÕES ARITMÉTICAS

Principais Funções Aritméticas em PHP Moderno.


Arredondamento para cima: ceil( )

Exemplos:

ceil(56.7) = 57

ceil(11.2) = 12

Observação: O arredondamento se dará para cima em qualquer situação entre os valores de 0.1 a 0.9.




Arredondamento para baixo: floor( )

Exemplos:

floor(56.7) = 56

floor(11.2) = 11

Observação: O arredondamento se dará para baixo em qualquer situação entre os valores de 0.1 a 0.9.




Arredondamento com precisão: round( )

Exemplos:

round(56.7) = 57

round(56.76, 1) = 56.8

Observação: O primeiro elemento corresponde ao VALOR NUMÉRICO e o segundo elemento às CASAS DECIMAIS a serem arredondadas. Se não informar as casas decimais, o arredondamento será para o limite mais próximo.


Raiz quadrada: sqrt( )

Exemplos:

sqrt(36) = 6

sqrt(81) = 9

sqrt(144) = 12

sqrt(42) = 6.4807406984079


Divisão inteira: intdiv( )

Exemplos:

intdiv(7, 2) = 3

intdiv(17.54, 2) = 8

Observação: O resultado virá num número inteiro descartando os valores após a vírgula. O primeiro elemento é o número e o segundo elemento é o divisor.

Chamando o número PI: pi( )

Exemplos:

pi() = 3.1415926535898

M_PI = 3.1415926535898

Observação: M_PI é uma constante e não uma função, mas também pode ser usado para chamar o valor de PI.


Máximo de uma sequência: max( )

Exemplos:

max(9, 12) = 12

max(3, 6, 89, 3, 36, 27, 5, 1, 71) = 89

Observação: Será retornado o maior valor da sequência de números.


Mínimo de uma sequência: min( )

Exemplos:

min(9, 12) = 9

min(3, 6, 89, 3, 36, 27, 5, 1, 71) = 1

Observação: Será retornado o menor valor da sequência de números.


Potência: pow( )

Exemplos:

pow(5, 2) = 25

pow(2, 3) = 8

Observação: O primeiro elemento é o número e o segundo elemento é a potência a qual o número deverá ser elevado.


Valor absoluto: abs( )

Exemplos:

abs(5.5) = 5.5

abs(-21) = 21

abs(-0.052) = 0.052

abs(053.47) = 53.47







Conversão de bases: base_convert( )

Exemplos:

base_convert(2143, 10, 16) = 85f

base_convert(42, 8, 10) = 34

base_convert(f2, 16, 2) = 11110010

Observação: O primeiro elemento é o número, o segundo elemento é a base numérica desse número e o terceiro elemento é a base numérica a ser convertido esse número (Decimal=10 / Binário=2 / Octal=8 / Hexadecimal=16).

Hipotenusa: hypot( )

Exemplos:

hypot(9, 12) = 15

hypot(27, 36) = 45

Observação: O primeiro elemento corresponde ao CATETO OPOSTO e o segundo elemento ao CATETO ADJASCENTE.






Seno: sin( )

Exemplos:

sin(30) = -0.9880316241

sin(45) = 0.85090352453412

sin(60) = -0.30481062110222

sin(90) = 0.89399666360056

Observação: Retorna como resultado o seno (em radianos) de um determinado ângulo em radianos.

Cosseno: cos( )

Exemplos:

cos(30) = 0.15425144988758

cos(45) = 0.52532198881773

cos(60) = -0.95241298041516

cos(90) = -0.44807361612917

Observação: Retorna como resultado o cosseno (em radianos) de um determinado ângulo em radianos.

Tangente: tan( )

Exemplos:

tan(30) = -6.4053311966463

tan(45) = 1.6197751905439

tan(60) = 0.32004038937956

tan(90) = -1.9952004122082

Observação: Retorna como resultado a tangente (em radianos) de um determinado ângulo em radianos.
